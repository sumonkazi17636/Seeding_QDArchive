Data for:  Energy democracy in northeastern North America
Please cite any data used from this data project as follows:
Burke, Matthew. 2018. "Energy democracy in northeastern North America". Qualitative Data Repository. https://doi.org/10.5064/F6BUAX58

This data project has been initiated for research on energy democracy initiatives (EDI) and their transition narratives, and more broadly on social-ecological-technical systems related to renewable energy transition in the region of northeastern North America to strengthen initiative-based practice and learning and support diverse and participatory analytical approaches. Documentation and metadata for this research include contextual information about the study, research methods used, variable definitions, units of analysis, format and file type of the data, a description of the data capture and collection methods, data source inventory, explanation of data coding and analysis performed and details of who has worked on the project and performed each task. A Methods and Data Management Plan is used to organize the documentation and metadata needed to support this research.


Standard access for all files listed below except Burke_EDI_NENA.NVivoProject.2017.nvp. This file (a full NVivo package) includes full-text materials with potential copyright constraints and can be made accessible on a limited one-on-one basis only to secondary users who have a verified academic affiliation and present a brief research plan specifying why they need access and how they plan to use the materials in the file. If you would like to request access, please do so via the catalog interface and be prepared to present those two pieces of information.

DATA PROCESSING NOTE

The data were processed according to the Qualitative Data Repository's curation policy. All files were checked for integrity. Files were checked to make sure they match documentation and consistent file names were applied. Internet hyperlinks were archived using perma.cc and original links and perma.cc links are available in documentation files (0_Burke_EDI_NENA.inventory.text.2017.10.08.csv , 0_Burke_EDI_NENA.inventory.text.2017.10.08.ods , 0_Burke_EDI_NENA.inventory.text.2017.10.08.tab).  Permanent links were added by QDR on May 30, 2018.

Files were converted from PDF formats to PDF/A using Adobe Acrobat Pro DC Version 2018.009.20050, XMP tags edited using AutoMedata version 1.0.0.20.
