**Title**
A Constructed Peace: Appendices

**Suggested Citation**
Please cite any data used from this data project as follows: 

Trachtenberg, Marc. 2020. "A Constructed Peace: Appendices". Qualitative Data Repository. https://doi.org/10.5064/F6B9CGIV

**Description**

This data collection - The Trachtenberg Papers - broadly concerns Cold War policy from the end of WWII to 1964. The data was accumulated in order to write several books and articles relating to Cold War relations during this pivotal period, most notably A Constructed Peace: The Making of the European Settlement, 1945-1963 (Princeton: Princeton University Press, February 1999).

This particular data project contains eight appendices that deal either with matters of detail or with issues tangentially related to the main argument in that book. They began their lives as footnotes in the manuscript, but eventually grew so large and unwieldy that I made them into appendices. When the manuscript as a whole had to be cut (I was given a 235,000-word limit), I decided to move them into the internet supplement on the UCLA website. They are now archived in this collection on QDR.

The reference system used in the appendices (short titles and all) is the same as the one used in the book and is explained there in the "Sources and Bibliography" section.

**Data Processing Note**

The data were processed according to the Qualitative Data Repository’s curation policy. All files were checked for integrity. Files were checked to make sure they match documentation and consistent file names were applied. Documentation with file descriptions and original as well as archived URLs was generated and can be found in Trachtenberg_Appendices_Master.xlsx. Word files were converted to PDF and PDF/A using Adobe Acrobat Pro DC Version 2020.006.20034, XMP tags edited using AutoMedata version 1.0.0.20.
