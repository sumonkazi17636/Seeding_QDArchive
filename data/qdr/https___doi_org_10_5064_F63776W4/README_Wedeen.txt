Project Summary: This project is part of a broader inquiry into neoliberal autocracy and its unmaking through the case study of the Syrian regime under Bashar al-Asad. Analyzing both the tenacity and incompleteness of ideological reproduction, the author takes as emblematic the work of Allayth Hajju, one of Syria’s best-known television directors. His comedy illustrates and helps perpetuate the ideology of neoliberal autocracy; at the same time, it provides some openings for, while attempting to manage, an oppositional consciousness

Data Abstract: The project contains three forms of data:
1. Selected video-clips from the comedy series with English subtitles, were collected on fieldtrips to Syria in 2010 and 2011, and subsequent trips to Lebanon after the beginning of the Syrian conflict. The author also spent the winter of 2012 in Paris, where she conversed repeatedly with displaced Syrians.
2. Statistical materials underlying the appendix to chapter 1 of the book
3. A list of all URLs used in the book with archived versions of the links in perma.cc, webrecorder.io, or the Internet Archive

DATA PROCESSING NOTE

The data were processed according to the Qualitative Data Repository’s curation policy. All files were checked for integrity. Files were checked to make sure they match documentation and consistent file names were applied. URLs were archived using perma.cc where possible and original links and perma.cc links are available in documentation files (Wedeen_archivedurls_bibliography.xlsx, Wedeen_archivedurls_notes.xlsx). Videos were archived using webrecorder.io where possible and original links and webrecorder.io links are available in a separate documentation file (Wedeen_archivedurls_videos.xlsx). Files were converted from PDF formats to PDF/A using Adobe Acrobat Pro DC Version 2019.010.20098, XMP tags edited using AutoMedata version 1.0.0.20.
