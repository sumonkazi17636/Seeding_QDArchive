Adverse psychological effects amongst self-tested individuals living in couples in urban Blantyre, Malawi

Please cite any data used from this data project as follows:

Kumwenda, Moses. 2019. "Adverse psychological effects amongst self-tested individuals living in couples in urban Blantyre, Malawi". Qualitative Data Repository. https://doi.org/10.5064/F6RL3PS2.

We investigated adverse psychological effects and coping mechanisms following HIV-positive and HIV-discordant test results amongst self-tested individuals living in couples in urban Blantyre, Malawi. 35 qualitative in-depth interviews with self-tested individuals living in couples were conducted.

DATA PROCESSING NOTE

The data were processed according to the Qualitative Data Repository's curation policy. All files were checked for integrity. Files were checked to make sure they match documentation and consistent file names were applied. QDR conducted basic de-identification checks using automated translation from Chichewa. Files were converted to PDF/A formats using Adobe Acrobat Pro DC Version 2018.009.20050, XMP tags edited using AutoMedata version 1.0.0.20.
